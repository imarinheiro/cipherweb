Funções do RSA

Calcular N

O algoritmo de calcular N utiliza como entrada dois números primos (P, Q) e gera como saída o resultado N. Esse resultado é utilizado como parte das chaves privada e pública. A fórmula para gerar a o N é:

N ⁼ P × Q

Calcular Tociente Φ

O algoritmo para calcular o Tociente de Euler utiliza como entrada dois números primos (P, Q). Esse cálculo utiliza a seguinte fórmula:

Φ ⁼ (P - 1) × (Q - 1)

Selecionar E

O algoritmo para selecionar E utiliza como entrada o Tocient Φ. As condições para a seleção do número são:

  1. Ser relativamente primo
  2. Ser maior ou igual a 3
  3. Ser menor que Tocient Φ

Para que dois numeros sejam relativamente primos, eles não devem compartilhar fatores comuns, ou seja, não deve existir um número inteiro (exceto 1) que divida ambos os números.

Calcular D

O algoritmo para calcular D utiliza como entrada os parâmetros E e Tociente Φ. Esse cálculo obtem como saída o valor D, tal que:

(E × D) mod Φ ≡ 1

O valor de D deve possuir o mesmo sinal de Tociente Φ. Pode-se dizer que E e D são inversos multiplicativos de mod Φ, ou seja:

E × D ≡ 1 mod Φ

D ≡ E-1 mod Φ

Transformar Texto Livre para Inteiro

O algoritmo para criptografar texto utiliza a tabela ASCII "American Standard Code for Information Interchange" para transformação do caractere para o seu referente código em decimal.

Para realizar uma busca do código referente basta acessar a página Mapa ASCII do W3Schools.

Criptografar

O algoritmo para criptografar utiliza 3 parâmetros como entrada: (M, N e E). M é a Mensagem (texto livre ou o número) que deseja criptografar. Os parâmetros N e E são partes calculada para a chave pública. A saída é o texto criptografado denominado C. A fórmula para realizar a criptografia é a seguinte:

C ⁼ ME mod N

Descriptografar

O algoritmo para descriptografar utiliza 3 parâmetros como entrada: (C, N e D). C é a Mensagem Criptografada (texto ou número) que deseja descriptografar. Os parâmetros N e D são partes calculada para a chave privada. A saída é mensagem descriptografado denominado M. A fórmula para realizar a descriptografia é a seguinte:

M ⁼ CD mod N
